### Installation
1. Clone the repo 
```sh
git clone https://github.com/SalmaaOsamaa/React-Food-Ordering-App.git
```

2. install dependenscies
 ```bash
   npm install
  ```
   
3. run the project 
```bash
   npm start
   ```
